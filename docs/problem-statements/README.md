# Problem Statements

Write one markdown file per research target.

Each file should answer:

- What is the exact question?
- What is known already?
- What would count as progress?
- What assumptions are being made?
- What is the smallest meaningful experiment?

Suggested structure:

```md
# Problem Title

## Statement

## Background

## Constraints

## Candidate Approaches

## Open Questions

## Next Experiment
```
